# EdgeRouter Feature Wizards
This repo contains several feature wizards for EdgeRouter's EdgeOS.

In order to add any wizard:
1. Download the Wizard's tar.gz file
2. Log into your router
3. Navigate to the Wizards page
4. Press on the + near "Feature Wizards"
5. Enter the Wizard's name and upload its file

## System Log Viewer
This Feature Wizard allow to view the last 1000 lines of the primary system log file (/var/log/messages).
This log file contains a historical record of all processes, system events, errors, warnings, and messages generated by the operating system and services running on the device.

## Dnsmasq Log Viewer
This Feature Wizard allow to view the last 1000 lines of the primary system log file (/var/log/dnsmasq.log).
This log file contains logging information related to the DNS and DHCP services provided by dnsmasq.
